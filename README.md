# Claude Code Game Studios (CCGS)

一个基于 AI 的游戏开发协作框架，覆盖从策划、设计、开发到测试的完整游戏开发流程。

## 快速开始

### 前置要求
- Git
- Python 3.8+

### 安装

git clone https://github.com/canglan211-hub/CCGS-project.git

## 项目结构

| 目录 | 说明 |
|------|------|
| .claude/agents/ | AI Agent 定义（程序、美术、QA 等 30+ 角色） |
| .claude/skills/ | 技能文件（代码审查、平衡检查、Bug 分类等） |
| docs/ | 文档和引擎参考 |
